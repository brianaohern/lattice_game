# Lattice Game

Development Branches & Workflow
----------------------

**Master**: Currently released branch

**Develop**: Development branch for preparing all features and releases

**Release** branches: Created from develop; Used to prepare a new release version; Merged into develop and master; Prefixed with 'release/'

**Feature** branches: Created from develop; Used to create new and modify existing features; Merged into develop only; Prefixed with 'feature/'

**Hotfix** branches: Created from master; Used to quickly fix bugs and issues that can't wait for a new release; Merged into develop and master; Prefixed with 'hotfix/'
